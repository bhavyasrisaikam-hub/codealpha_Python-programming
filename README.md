Python Programming Projects

This repository contains a collection of Python projects developed to strengthen programming, problem-solving, and automation skills. The projects demonstrate the practical application of Python concepts such as loops, functions, conditional statements, file handling, and user interaction.

Projects

Hangman Game

A console-based word guessing game where the player attempts to guess a hidden word letter by letter within a limited number of attempts. This project focuses on string manipulation, loops, and logical thinking.

Stock Portfolio Tracker

A simple application that allows users to manage and track stock investments. It calculates portfolio value and provides an overview of investments, helping users understand basic data handling and calculations in Python.

Task Automation with Python Scripts

A set of automation scripts designed to perform repetitive tasks efficiently. This project demonstrates how Python can be used to improve productivity through automation and file operations.

Basic Chatbot

A rule-based chatbot that interacts with users through predefined responses. The chatbot handles simple conversations and showcases the use of conditional statements, functions, and loops.
